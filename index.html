📐 Probability Formulas Used

Classical Probability

P(A) = favorable / total

Bayes' Theorem

P(A|B) = P(B|A)×P(A) / P(B)

Normal Distribution

f(x) = (1/σ√2π) × e^(-(x-μ)²/2σ²)

Poisson Distribution

P(X=k) = (λ^k × e^(-λ)) / k!

Binomial Distribution

P(X=k) = C(n,k) × p^k × (1-p)^(n-k)

Conditional Probability

P(A|B) = P(A∩B) / P(B)

Z-Score

z = (x - μ) / σ

Variance

σ² = Σ(x - μ)² / n

Standard Deviation

σ = √(Σ(x - μ)² / n)

Percentile Rank

(below + 0.5×equal) / n × 100
